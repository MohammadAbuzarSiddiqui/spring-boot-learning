# Spring Boot Learning

This repository contains my practice projects while learning Spring Boot and backend development.

The projects are created while following the Spring Boot course by Chad Darby, covering core concepts and real-world backend development practices.

## Topics Covered

- Spring Boot fundamentals
- Dependency Injection
- Spring MVC
- REST API development
- Spring Data JPA and Hibernate
- CRUD applications
- Spring Security
- Database integration

## Project Structure

Each folder in this repository represents a separate module or project based on a specific concept from the course.

## Technologies Used

Java, Spring Boot, Maven, Hibernate, Spring Data JPA, REST APIs, MySQL/PostgreSQL, IntelliJ IDEA.
